Obelisk is a strongly typed object oriented programming language. Obelisk's type system intends to be a form of strongly typed 'duck typing'.

Contents:
1. Syntax

Syntax

Obelisk's syntax is described by this happy file.

Here is a description in mock-bnf.

obelisk: stmt obelisk

stmt: '(' stmtcomplex ')' | stmtexp

stmtcomplex: def | if | localset | globalset

def: 'def' vars '(' obelisk exp ')'

if: 'if' exp '(' obelisk exp ')' '(' obelisk exp ')'

localset: ':' var exp

stmtexp: exp

exp: number | var | '(' vars ')' | '(' exp operator exp ')' | 'true' | 'false'

vars: var vars

Here is an example, which defines a function called add:

(def add (x y) ((x + y)))
